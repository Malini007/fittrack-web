# FitTrack Web 🏃‍♀️

A Strava-inspired running tracker web application built with React and Material UI, connected to the FitTrack Spring Boot backend API.

## Features
- Beautiful dark-themed UI with glassmorphism design
- User registration and JWT-based login
- Log runs with route name, distance, duration and date
- Auto pace calculation (min/km)
- Personal stats dashboard — total runs, distance, avg pace, best run
- Interactive distance chart with Recharts
- Full run history with delete functionality

## Tech Stack
- React 18
- Material UI (MUI)
- Recharts for data visualization
- Axios for API calls
- React Router DOM

## Backend
This frontend connects to the FitTrack REST API:
👉 [github.com/Malini007/fittrack](https://github.com/Malini007/fittrack)

## Setup
1. Clone this repository
2. Run `npm install`
3. Make sure the FitTrack backend is running on `http://localhost:8080`
4. Run `npm start`
5. Open `http://localhost:3000`
